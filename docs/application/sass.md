# SASS について

SASSh は「記法」と「コンパイルしている言語」の 2 種類の組み合わせから構成される。

## 記法

SASS 記法と SCSS 記法が存在する。CSS 記法との互換性の問題から、SASS 記法はほとんど流通しなかったため、注記しない限り SCSS 記法を指す。

## 言語

現在（2023/3）、Dart、C 言語 / C++、Ruby の 3 種類が存在する。基本的に Ruby は使われない（サポート終了）。

Dart が公式が推奨している Sass。Dart Sass という名前。

C 言語/C++は現在公式は非推奨。Lib Sass という名前。こいつによく苦しめられる思い出がある。

# 改行文字

## 以下の 3 種類がある

- `\n`
  - LF(Line Feed)
  - 一般的なもの
- `\r\n`
  - CR + LF(Carriage Return + Line Feed)
  - Windows で使われる
- `\r`
  - CR(Carriage Return)
  - 古い Mac で使われていた(MacOS 9 以前)

また、`\\n`と書く場合もあるので注意.

# C++について

## 実行方法

コンパイルし、オブジェクトファイルを生成して、それを実行する。以下のコマンドでコンパイル → 実行が可能。

```sh
$ g++ -o ${OBJECT_FILE} ${CPP_FILE}.cpp
$ ./${OBJECT_FILE}
```

## 別々にコンパイルする

通常開発では、ファイルを分割する。その際に、併せてコンパイルするやり方は以下。

```sh
$ g++ -o ${OBJECT_FILE} ${A}.cpp ${B}.cpp
```

しかし、これだと両方のコンパイルをし直す必要があるので、片方ずつコンパイルする。分割コンパイルである。

```sh
$ g++ -c ${A}.cpp
$ g++ ${B}.cpp ${A}.o
```

## オプション

- o
  - オブジェクトファイル名指定
- I
  - インクルードディレクトリを追加する
- L
  - ライブラリパスを追加してリンクする
- lm
  - ライブラリをリンクする
